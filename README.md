# skills

[![skills.sh](https://skills.sh/b/hifizz/skills)](https://skills.sh/hifizz/skills)

> zilin（[@hifizz](https://github.com/hifizz)）的个人 [Agent Skills](https://agentskills.io) 库 —— 把日常开发中真实在用的能力，一步步沉淀成可安装、可分享的 skill。博客：<https://zilin.im>

## 安装

```bash
# 安装全部 skill
npx skills add hifizz/skills

# 安装指定 skill
npx skills add hifizz/skills --skill <skill-name>
```

支持 Claude Code、Cursor、Codex、OpenCode 等 50+ agent（遵循 agentskills.io 开放标准）。
